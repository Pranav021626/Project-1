About This Project

Welcome to the Phone Number Details Website! This project is built using Django and Python, along with HTML and CSS for a clean, responsive interface. It allows users to enter a phone number and instantly get useful details such as the country, state, carrier, timezone, and validity of the number.

📌 Features

  • Enter any phone number in international format (e.g., +919876543210)
  • Get country and state/region information
  • Check carrier (like Airtel, Jio, Vodafone, etc.)
  • See timezone of the number
  • Validate if the phone number is correct

💡 Technology Used

  • Python programming language
  • Django framework for backend and routing
  • HTML5 and CSS3 for frontend layout
  • Phonenumbers Python library for number parsing and details
  • Responsive design with clean, readable UI
